## GET the SQL file for project
here you can also get an .sql file 
Go to Localhost/phpmyadmin in your web browser 
and create a database name artify 

 ```sql
   CREATE DATABASE artify
``` 
 
URL:-

    http://localhost/phpmyadmin/
